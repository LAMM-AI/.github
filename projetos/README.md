# Projetos

Este diretório guarda referências de projetos do laboratório.

- `planos/`: PDFs locais de planos de trabalho (**não versionados**; `*.pdf` está no `.gitignore`).
- `producoes/`: PDFs locais de produções (**não versionados**; `*.pdf` está no `.gitignore`).

Se precisar compartilhar PDFs publicamente, use Git LFS, GitHub Releases, ou um link externo (Drive).
