# Suporte

## Onde pedir ajuda
Dependendo do repositório, estas são as rotas recomendadas:

- **Bugs e tarefas rastreáveis:** abra uma Issue (use os templates).
- **Ideias / propostas:** abra uma Issue do tipo *Feature request*.
- **Dúvidas gerais:** abra uma Issue do tipo *Pergunta / suporte* (ou Discussions, se o repositório tiver).

## Antes de pedir ajuda
- Forneça contexto e objetivo.
- Inclua logs/prints quando fizer sentido (remova informações sensíveis).
- Descreva seu ambiente (SO, versão, hardware, etc.) quando relevante.
