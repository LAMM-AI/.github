# LEMM.IA — Handbook (templates e políticas)

Este diretório concentra **templates prontos para copiar** e **políticas mínimas** para padronizar repositórios da organização, com foco em:

- formação (IC/TCC/Mestrado),
- reprodutibilidade,
- e caminho até artigo (paper + evidências).

## Como usar (novo projeto)

1. Crie um novo repositório do projeto na organização.
2. Copie o conteúdo de [`PROJECT_TEMPLATE/`](PROJECT_TEMPLATE/) para a raiz do novo repositório.
3. Ajuste `README.md`, `CITATION.cff`, `requirements.txt/environment.yml` e a estrutura conforme o projeto.
4. Use os templates em [`TEMPLATES/`](TEMPLATES/) para padronizar experimentos e updates.

## O que está aqui

- Políticas: `GPU_POLICY.md`, `DATA_POLICY.md`, `REPRO_CHECKLIST.md`, `AUTHORSHIP_POLICY.md`, `SECURITY_BASICS.md`
- Template de repositório: [`PROJECT_TEMPLATE/`](PROJECT_TEMPLATE/)
- Templates operacionais: [`TEMPLATES/`](TEMPLATES/)
